
# **Project Blueprint: JC 행운번호 추첨기**

## **Overview**

This document outlines the design and development plan for a simple, modern, and visually appealing JC 행운번호 추첨기 web application. The application will be built using framework-less web technologies (HTML, CSS, JavaScript) and will adhere to modern web standards.

## **Core Features**

*   **Number Generation:** Generate a set of 6 unique random numbers between 1 and 45.
*   **Display:** Display the generated numbers in a clear and visually appealing way.
*   **User Interaction:** Allow the user to generate a new set of numbers by clicking a button.
*   **Responsive Design:** Ensure the application looks and works well on all screen sizes, from mobile phones to desktop computers.

## **Design and Styling**

*   **Layout:** A clean and simple layout with a central focus on the generated numbers.
*   **Color Palette:** A modern and energetic color palette.
*   **Typography:** Clear and readable fonts.
*   **Visual Effects:** Subtle animations and shadows to enhance the user experience.

## **Implementation Plan**

1.  **HTML (`index.html`):**
    *   Set up the basic structure of the page, including a title, a container for the numbers, and a "Generate" button.
2.  **CSS (`style.css`):**
    *   Style the number container, the individual numbers, and the button.
    *   Implement a responsive layout using media queries.
3.  **JavaScript (`main.js`):**
    *   Create a function to generate an array of 6 unique random numbers between 1 and 45.
    *   Create a function to display the numbers in the HTML.
    *   Add an event listener to the "Generate" button to trigger the number generation and display.
